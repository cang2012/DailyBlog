# 从官网安装

- 在官网注册
- $ ./ngrok authtoken your_token
- 查看帮助：$ ./ngrok help
- 开始：$ ./ngrok http 80  这里的80端口是应用程序的端口
- 把安装ngrok的目录放在环境变量中
- 管理员打开命令窗口：ngrok version
