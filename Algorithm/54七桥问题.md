```
───通路：顶点与边交替出现  
│   │   
│   └───简单通路：所有边不相同
│   |___初级通路：所有顶点不相同
│   |___连通图：任意两个顶点都是连通的  
│   |___欧拉通路，欧拉迹：经过每条边只有一次  
│   
└───回路：起点和终点相同
    │___简单回路：所有边不相同
    │___初级回路：所有顶点不相同
    |___欧拉回路：每条边通过仅仅一次的贿赂
    |___欧拉图：具有欧拉回路的图  
```
